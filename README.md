Orașul Tău - Galați
   Orașul Tău este o platformă digitală modernă menită să apropie cetățenii de administrația locală. Proiectul oferă o soluție simplă pentru raportarea problemelor urbane, propunerea de idei noi și monitorizarea dezvoltării orașului Galați în timp real.

Funcționalități Principale
 - Hartă Interactivă: Vizualizezi toate sesizările din oraș direct pe hartă, colorate în funcție de categoria lor.

 - Sistem de Sesizări: Poți raporta rapid probleme precum infrastructura deteriorată, gunoiul sau iluminatul stradal defect.

 - Forum de Propuneri: O secțiune dedicată unde poți veni cu idei pentru îmbunătățirea orașului sau a site-ului. Comunitatea poate vota și comenta aceste idei.

 - Accesibilitate Avansată: Un widget special care permite ajustarea textului, contrastului și a fontului pentru persoanele cu dizabilități vizuale sau dislexie.

 - Panou Admin: Administrația poate schimba statusul sesizărilor (În lucru, Rezolvat, Respins) și poate oferi răspunsuri oficiale.

Tehnologii Folosite
 - Frontend: HTML5, CSS3 (Modern UI), JavaScript (ES6+).

Hartă: 
 - Leaflet.js și OpenStreetMap.

Iconițe: 
 - Lucide Icons.

Backend: 
 - Node.js cu API-uri pentru gestionarea rapoartelor, voturilor și comentariilor.

Bază de date: 
 - MongoDB (pentru stocarea permanentă a datelor).

Cum pornești proiectul
 - Descarcă sau clonează folderul proiectului.

 - Asigură-te că ai Node.js instalat.

 - Instalează dependențele (dacă există) folosind npm install.

 - Pornește serverul și deschide index.html în browser.

Obiectivele Proiectului
  Scopul acestui proiect este transparența. Ne dorim ca fiecare cetățean să simtă că vocea lui contează și că problemele raportate sunt luate în seamă de autorități, transformând Galațiul într-un oraș mai inteligent și mai prietenos.
